# Samsung-Retail-Sales-Analysis
Retail sales data analysis and visualization project using Python and Tableau, focused on Samsung products.

# Project Overview
This project focuses on analyzing a retail forecasting dataset for Samsung products. The goal was to clean, transform, and analyze the data to uncover insights that can support business decision-making.

# Data Cleaning
The dataset contained several inconsistencies which were handled using Python:
Missing values were identified and treated
Duplicate records were removed
Outliers were detected and handled
Data types were corrected for analysis

# Tools & Technologies
Python (Pandas, NumPy)
Tableau
Jupyter Notebook / PyCharm

# Data Analysis
After cleaning, the data was aggregated and analyzed to answer key business questions:
Which products generate the most revenue?
Which regions perform best?
How do discounts affect sales?
What are the sales trends over time?
In what quarter is sales at its peak?
What country makes the most sales?
What city makes the most sales?
What sales channel was most used?
What payment method was most used?

# Visualization
An interactive dashboard was created in Tableau to present insights:
Sales trends over time
Revenue by region and country
Top-performing products
Sales channel distribution
Customer ratings analysis
